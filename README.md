# Balloon Popper

A small 2D Balloon Popper Game made with Unity Engine.

---

## Game Overview

- **Engine:** Unity 2022.3.41 LTS  
- **Genre:** Casual  
- **Platform:** WEBGL (initially)  
- **Status:** Deployed to Itch.IO  

Pop the balloons before they escape!
Each balloon gives you points. 
How high can you score? Let's see....
